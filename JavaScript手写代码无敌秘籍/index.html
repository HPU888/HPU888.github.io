1. 实现一个 new 操作符

new 操作符做了这些事：

  • 它创建了一个全新的对象。
  • 它会被执行 [[Prototype]] （也就是 proto ）链接。
  • 它使 this 指向新创建的对象。
  • 通过new创建的每个对象将最终被[[Prototype]]链接到这个函数的prototype对象上。
